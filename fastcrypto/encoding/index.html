Module fastcrypto::encoding

source ·
Expand description

Encodings of binary data such as Base64 and Hex.

Example

assert_eq!(Hex::encode("Hello world!"), "0x48656c6c6f20776f726c6421");
assert_eq!(Base64::encode("Hello world!"), "SGVsbG8gd29ybGQh");
