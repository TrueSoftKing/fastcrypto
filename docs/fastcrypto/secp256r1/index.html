Module fastcrypto::secp256r1

source ·
Expand description

This module contains an implementation of the ECDSA signature scheme over the secp256r1 NIST-P1 curve. The nonce is generated deterministically according to RFC6979.

Messages can be signed and the signature can be verified again:

Example

use rand::thread_rng;
let kp = Secp256r1KeyPair::generate(&mut thread_rng());
let message: &[u8] = b"Hello, world!";
let signature = kp.sign(message);
assert!(kp.public().verify(message, &signature).is_ok());
